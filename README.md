## Running a Project Locally

1.   Clone a Project

```bash
git clone https://github.com/umerinayat/news-app.git
```

2. Go to Project Root directory

```bash
cd ./news-app
```

3. Make sure you have docker installed in a system and run the following docker commands in 

```bash
docker-compose up
```
